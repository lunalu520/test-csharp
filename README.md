# test-csharp
Technical task
## Overview
This project is a collection of small tasks, each implemented in a separate class for clarity and organization.

## Tasks
1. Task1NumberCheck.cs  
   If the entered number is greater than 7, then prints “Hello”.

2. Task2NameCheck.cs  
   If the entered name matches “John”, then outputs “Hello, John”, if not, then outputs "There is no such name".

3. Task3ArrayCheck.cs  
   Accepts an array of numbers and outputs array elements that are multiples of 3.

4. Task4BracketSequenceAnswers.cs
   Validates a bracket sequence:  
   - Prints whether the sequence is correct.  
   - If not, suggests what needs to be fixed.

5. Program.cs  
   Entry point that runs all tasks.

## How to Run
1. Open the project in Visual Studio or VS Code.  
2. Build and run the console application.  
3. Follow the instructions in the console.
